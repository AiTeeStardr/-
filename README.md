# เครื่องคิดเลขค่าไฟฟ้า


เว็บแอปพลิเคชันสำหรับคำนวณค่าไฟฟ้าแบบอัตราก้าวหน้า ประเภทที่ 1 บ้านอยู่อาศัย สร้างด้วย HTML, CSS (Vanilla) และ JavaScript (Vanilla)

## การทำงานของโปรแกรม
1. **รับข้อมูล (Input):** ผู้ใช้กรอก **จำนวนหน่วยไฟฟ้า (kWh)** ที่ใช้งานลงในช่องรับข้อมูล แล้วกดปุ่ม "คำนวณค่าไฟ"
2. **ประมวลผล (Process):** 
   - โปรแกรมจะนำจำนวนหน่วยที่ได้ไปคำนวณตาม **อัตราค่าไฟแบบก้าวหน้า** (ยิ่งใช้เยอะ ราคาต่อหน่วยในขั้นถัดไปจะยิ่งสูงขึ้น)
   - แยกการคิดเงินออกเป็นช่วงๆ (เช่น 15 หน่วยแรก, 10 หน่วยถัดไป, เป็นต้น)
   - นำค่าไฟฐานมารวมกับ **ค่าบริการรายเดือน** (38.22 บาท) และ **ค่า Ft** (0.1623 บาท/หน่วย)
   - นำยอดรวมทั้งหมดมาคำนวณ **ภาษีมูลค่าเพิ่ม (VAT 7%)**
3. **แสดงผล (Output):** 
   - แสดงตารางแจกแจงรายละเอียดการคำนวณในแต่ละช่วงหน่วยอย่างชัดเจน
   - แสดงสรุปยอดรวมที่ต้องชำระสุทธิ (รวม VAT แล้ว) ให้ผู้ใช้ทราบ
